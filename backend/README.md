# Backend 

#### Setup:
Remember to activate Python's local virtual environment: `$venv/Scripts/Activate.bat`
#
#### To Install:
After activating Python's virtual environment: `$pip install -t lib -r requirements.txt`
#
#### To Run:  
`$cd src`  
If using windows: run `windows_run.bat`  
Else run `run.bat`
#
#### To Test:
`$pytest`
